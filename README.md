# 3T3-L1-RNA-Sequencing-Transcriptomics
These are codes used for my PhD project to analyse the 16s RNA seqeuncing output. From raw data (paired reads), Deseq2, and generation of visual plots by using Windows Powershell/Bash & R.
RNA-seq analysis pipeline for IL-13 rs20541 study

Steps:
1. Quality control (FastQC)
2. Quantification (Salmon)
3. Differential expression (DESeq2)
4. KEGG enrichment (clusterProfiler)
5. Visualization (ggplot2, pheatmap)

Input: FASTQ files (SRA: TBU)
